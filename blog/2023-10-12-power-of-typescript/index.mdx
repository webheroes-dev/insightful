## Discriminated union

- usefull to let TS infer function conditional return type

for example below:
- status is either `'ok'` or `'error'`
- if ok, `data` is list of strings
- in case of error, `data` is `null` and has `error` message

before

```ts showLineNumbers
type Result = {
    // highlight-start
    status: 'ok'
    data: string[]
    // highlight-end
} | {
    // highlight-start
    status: 'error'
    data: null
    error: string
    // highlight-end
}

function do(): Result {
  // ...
}

const result = do()

if (result.status === 'ok') {

}
```
